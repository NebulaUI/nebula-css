Nebula CSS — Demo

Github Repository & Documentation

ITCSS and BEMIT based Sass/CSS framework that is ultra exensible and scales to any project size.

You are encouraged to use your browser dev tools to see what markup structure these components require

Note: Anything prefixed with demo- is for demonstration purposes only and does not ship with the framework

Install via NPM

npm install --save nebula-css

Flag

Throws image-like and text content side by side vertically aligning the two items. The dimensions of flag component are dictated by the space occupied by it's descendants. The flag body will take up all the remaining space within the flag's parent container.

The flag is useful in that you can mix fix width components along with fluid components and it also provides vertical alignment of the content.

Image-like content could consist of:

  • An image
  • A button
  • A video
  • Anything that takes up abitrary space
